1.Была реализована комнатная система.
один игрок начинает (Public match), другие игроки подключаются к его комнате во вкладке(Join).
2.Так же игрок может начать приватную игру, в таком случае игрокам нужно будет подключаться 
по IP хоста(IP можно посмотреть в игровом окне, нажав кнопку"Назад").
Другие игроки заходят во вкладку (Join), и в поле с вводом IP, вводят IP хоста, после этого нажимая кнопку (Connect).
